# РУМТИБЕТ (Angular Simulator)

Одностраничное веб-приложение туристического агентства, разработанное в рамках производственной практики. Позволяет искать туры, просматривать услуги, взаимодействовать с интерактивными элементами (таймер, счётчик, живой ввод). Реализована двусторонняя привязка данных, валидация форм, анимации и экран загрузки.

## Стек
- **Frontend:** Angular 16+ (standalone), TypeScript, SCSS
- **Хранение данных:** LocalStorage, универсальный класс Collection\<T\>
- **Деплой:** Vercel

## Запуск локально
1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите сервер разработки: `ng serve`
4. Откройте в браузере `http://localhost:4200/`

## Деплой
Приложение развёрнуто на Vercel: [angular-simulator.vercel.app](https://angular-simulator.vercel.app/)
